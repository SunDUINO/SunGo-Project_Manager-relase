## 🇺🇸 Overview

**SunGo Project Manager** is a comprehensive extension for Visual Studio Code that supports the full Go project lifecycle—from rapid project scaffolding to building, testing, and advanced performance analysis. It is designed with developer convenience and code organization in mind.

### ✨ Key Features
* **Project Scaffolding:** Create Go structures using remote Git templates.
* **CGO Cross-Compilation:** Built-in support for Windows, Linux, and macOS.
* **Advanced Profiler:** Integrated `pprof` panel for CPU and Memory analysis.
* **Gopher Assistant:** Real-time feedback and Go-specific tips.

## 🇵🇱 Opis projektu

**SunGo Project Manager** to kompleksowe rozszerzenie dla Visual Studio Code, które wspiera pełny cykl życia projektu w języku Go – od błyskawicznego tworzenia struktury, przez budowanie i testowanie, aż po zaawansowaną analizę wydajności. Zostało stworzone z myślą o wygodzie i porządku w kodzie.

### ✨ Kluczowe funkcje
* **Szablony projektów:** Tworzenie struktury Go z wykorzystaniem szablonów Git.
* **Kompilacja Skrośna CGO:** Wbudowane wsparcie dla Windows, Linux i macOS.
* **Zaawansowany Profiler:** Zintegrowany panel `pprof` do analizy CPU i pamięci.
* **Gopher Assistant:** Wsparcie w czasie rzeczywistym i wskazówki dla programisty.
